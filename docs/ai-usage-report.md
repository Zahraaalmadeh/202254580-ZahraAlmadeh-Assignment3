# AI Usage Report

## 1. Tools Used and Use Cases

### ChatGPT
I used ChatGPT as the main AI tool during this assignment. It supported me in several stages of development, including:
- improving the structure and design of the portfolio website
- refining JavaScript features such as filtering, sorting, and validation
- helping integrate the GitHub API
- suggesting stronger UI/UX improvements for a more professional result
- helping generate and improve documentation files such as the README and technical documentation

### GitHub Copilot
GitHub Copilot can also be used for code completion and small coding suggestions during implementation. In this project, the main AI assistance was focused through ChatGPT for feature planning, debugging, and documentation support.

---

## 2. Benefits of Using AI
Using AI tools helped me:
- generate ideas faster
- improve code quality and organization
- better understand how to implement filtering, sorting, state management, and API integration
- write clearer and more complete documentation
- refine the interface into a more polished and user-friendly portfolio

AI was especially helpful in turning a simpler portfolio into a stronger Assignment 3 submission that better matches the rubric.

---

## 3. Challenges, Risks, and Limitations of AI Usage
Although AI was helpful, it also introduced some risks and limitations:

- AI suggestions are not always fully correct and may still contain logic or validation issues
- some generated code needed to be adjusted to match my exact project structure and real project data
- AI-generated text can sound generic if it is copied directly without revision
- over-reliance on AI can reduce independent problem-solving if it is used without understanding
- some examples may appear correct at first but fail in edge cases, such as email validation or UI behavior

Because of these risks, I reviewed and tested all AI-assisted code before using it in the final project.

---

## 4. Learning Outcomes
By using AI during this assignment, I learned:
- how to combine filtering and sorting in JavaScript
- how to use `fetch()` with an external API
- how to improve validation logic and user feedback
- how to manage persistent UI state using `localStorage`
- how to improve documentation quality and structure
- how to revise AI-generated suggestions into a result that better reflects my own work

AI also helped me become more confident in debugging and improving front-end code step by step.

---

## 5. Responsible Use and Modifications
I used AI responsibly by:
- reviewing every suggestion before applying it
- modifying AI-generated code to fit my real projects and assignment requirements
- testing the functionality manually after each major change
- rewriting and adapting text so the final submission reflects my own project and understanding
- using AI as a support tool for learning, debugging, and refinement, not as a substitute for understanding

I did not submit raw AI output without checking it. Instead, I used AI to support my development process while making sure the final work remained my own.

---

## 6. Conclusion
AI tools played an important supporting role in this assignment. They helped me improve both the technical implementation and the documentation, but I remained responsible for understanding, testing, and adapting the final work. This process helped me learn more effectively while maintaining academic integrity.
